Caso semplice di connessione in rete con socket tra client e server
Un solo client e un solo server, ma si può estendere a più client dato che ho fatto il codice più generale possibile
Il client manda il suo nome (preso da terminale) e il server risponde con "Ciao 'nome'!"

Si può introdurre il multi-threading per gestire ogni nuovo client con un sotto-processo del server
